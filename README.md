# aospdl

脚本一键设置国内镜像下载aosp源码

# 使用方法

直接按顺序输入以下命令即可开始下载aosp

使用国内镜像为  https://gerrit-googlesource.lug.ustc.edu.cn/git-repo

repo目录位置为 用户目录/bin/repo

aosp目录位置为 用户目录/aosp

```bash
curl https://raw.githubusercontent.com/myth-china/aospdl/main/aospdl.sh > aospdl.sh

chmod a+x aospdl.sh

./aospdl.sh
```
